CineSenseAI is an **AI-driven movie recommendation system** that improves traditional recommendation accuracy by **27-31%** through advanced **NLP, cosine similarity, and sentiment analysis**. The system delivers personalized suggestions based on **user preferences, reviews, and movie metadata**.

## Key Features

- **NLP-Based Recommendation Engine** ‚Äì Uses **TF-IDF vectorization and cosine similarity** to find similar movies.
- **Sentiment Analysis for Reviews** ‚Äì Analyzes audience reviews to improve movie rankings.
- **Real-Time Movie Data** ‚Äì Integrates with **TMDB API** for up-to-date information.
- **Dynamic User Interaction** ‚Äì Built with **Flask & AJAX** for a smooth and responsive UI.

## System Architecture

The platform consists of a **Flask backend**, a **JavaScript/AJAX-powered frontend**, and **a machine learning model** that processes user inputs to generate recommendations.

![System Architecture](/projects/movie-recommendation-architecture.png)

## Workflow

1. **User Input** ‚Äì Users enter a movie title or select preferences.
2. **Data Retrieval** ‚Äì The system fetches metadata from the **TMDB API**.
3. **Cosine Similarity Calculation** ‚Äì Finds movies with similar attributes.
4. **Sentiment-Based Ranking** ‚Äì Analyzes **user reviews and feedback** to refine recommendations.
5. **Personalized Movie Suggestions** ‚Äì Displays top recommended movies based on analysis.

## AI Models Used

- **TF-IDF Vectorization + Cosine Similarity** ‚Äì Measures movie similarity based on descriptions.
- **Sentiment Analysis Model** ‚Äì Uses **VADER (Valence Aware Dictionary and sEntiment Reasoner)** to assess movie reviews.
- **Collaborative Filtering (Future Expansion)** ‚Äì Enhancing recommendations based on user interactions.

## Technologies Used

- **Flask** ‚Äì Lightweight web framework for backend API.
- **AJAX** ‚Äì Enables real-time data fetching for a dynamic user experience.
- **TMDB API** ‚Äì Provides real-time movie details and metadata.
- **NLP & Machine Learning** ‚Äì Powers recommendation and sentiment analysis models.
